# Titanic EDA Project

## Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand passenger data, identify patterns, and analyze factors related to survival.

The main objective is to explore the dataset through data cleaning, visualization, and statistical analysis to gain insights into passenger survival.

---

## Dataset

The dataset contains information about Titanic passengers, including:

- Passenger class
- Gender
- Age
- Ticket fare
- Family relationships
- Embarkation location
- Survival status

The target variable is:

- **Survived**
  - 0: Not survived
  - 1: Survived

---

## Analysis Performed

- Data inspection and preprocessing
- Handling missing values
- Univariate analysis
- Categorical feature analysis
- Numerical feature distribution analysis
- Correlation analysis
- Feature relationship analysis with survival

---

## Visualizations

The project includes:

- Bar charts for categorical features
- Histograms for numerical features
- Correlation heatmap for numerical variables

These visualizations help identify patterns and relationships within the dataset.

---

## Key Findings

- Passenger gender showed a strong relationship with survival.
- Passenger class and fare were also important factors related to survival.
- Numerical features such as age and family-related variables showed weaker relationships.
